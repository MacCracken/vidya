# Code of Conduct

## Standard

We follow the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

## Summary

- Be respectful and constructive
- Focus on the work, not the person
- No harassment, trolling, or personal attacks
- Technical disagreements are welcome; make them about the code

## Enforcement

Report violations to: conduct@agnosticos.org
